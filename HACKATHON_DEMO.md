# 🚀 Agent League - Hackathon Demo Guide

## 🎯 **Demo Overview**
Agent League is a decentralized AI agent performance tracking and capital delegation platform built on Mantle Network. It enables users to discover, evaluate, and invest in top-performing AI trading agents.

---

## 🏗️ **Architecture Overview**

### **Smart Contracts (Hardhat Local)**
- **AgentRegistry**: Core registry for agent registration and performance tracking
- **AgentProfileNFT**: NFT-based agent profiles with metadata
- **DisputeResolver**: Decentralized dispute resolution system
- **AgentDelegation**: Capital delegation and reward distribution
- **Mock USDC**: Test stablecoin for demo purposes

### **Backend API (Node.js + TypeScript)**
- RESTful API serving agent data and performance metrics
- Real-time leaderboard updates
- Agent registration and management
- Oracle integration for price feeds

### **Frontend (React + TypeScript + Tailwind)**
- Modern 3D animated UI with Framer Motion
- Wallet connection (MetaMask)
- Interactive performance charts
- Agent registration interface
